# Miscellaneous

## DAVProvider
| DAVProvider     | URI                                                                  |                                   |
|-----------------|----------------------------------------------------------------------|-----------------------------------|
| FileProvider    | [`file:///your/path`](https://en.wikipedia.org/wiki/File_URI_scheme) | data is stored in the file system |
| MemoryProvider  | `memory:///`                                                         | data is stored in the memory      |
